Definition of all compiletime parameters for Index.

The Index class takes this as a template parameter.

Template Parameters

  • _coordinate_t ‚Äì type of coordinates, expected to be an unsigned int.

  • _val_t ‚Äì type of prefix sums, expected to be an unsigned int.

  • _D ‚Äì number of dimensions.

  • _class_key_t ‚Äì type of the dataset keys, expected to be an unsigned int.

  • _vec_generator ‚Äì generator object that implements the file and vec methods. See DiskVecGenerator.

  • _dependant_dim ‚Äì whether dimension 1 is dependant on dimension 0

  • _uniform_overlay_grid ‚Äì whether overlays are distributed uniformly (disables dependant_dim)

  • _orthotope_dims ‚Äì number or orthotope dimensions

  • _explain ‚Äì debugging parameter. Be verbose while creating and querying the index.

  • _progress_stream_t ‚Äì object that catches all the print output. See StdOutProgressStream.
