libSps.DiskPointsPrefixSum_2D

class libSps.DiskPointsPrefixSum_2D

Prefix sum index for points in 2-dimensional space.

add_point(self: libSps.DiskPointsPrefixSum_2D, pos: List[int[2]], desc: str = '') → None

Append a point to the data structure.

Parameters

  • pos (list[int[2]]) – The position of the point.

  • desc (str) – A description for the Point, defaults to “”.

The point will not be queryable until generate is called.

generate(self: libSps.DiskPointsPrefixSum_2D, from_points: int, to_points: int, verbosity: int = 1) → int

Generate a new dataset.

Parameters

  • from_points (int) – Index of the first point that shall be part of this dataset.

  • to_points (int) – Index of the last point that shall be part of this dataset.

  • verbosity (int) – Degree of verbosity while creating the dataset, defaults to 1.

Returns

The id of the generated dataset.

Return type

int

This may take a long time to compute.

Use len(index) to determine the index of the first and last point, as add_point may add multiple points per call.

This function is multithreaded.

count(self: libSps.DiskPointsPrefixSum_2D, dataset_id: int, from_pos: List[int[2]], to_pos: List[int[2]], verbosity: int = 0) → int

Count the number of points between from and to and in the given dataset.

Parameters

  • dataset_id (int) – The id of the dataset to query

  • from_pos (list[int[2]]) – The bottom left position of the query region.

  • to_pos (list[int[2]]) – The top right position of the query region.

  • verbosity (int) – Degree of verbosity while counting, defaults to 0.

Returns

The number of points in dataset_id between from_pos and to_pos.

Return type

int

to_pos must be larger equal than from_pos in each dimension.
