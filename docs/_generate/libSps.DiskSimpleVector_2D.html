class libSps.DiskSimpleVector_2D

Simple Vector for 2-dimensional orthotopes.

add_point(self: libSps.DiskSimpleVector_2D, start: List[int[2]], end: List[int[2]]) → None

Append an orthotope to the vector

Parameters

  • start (list[int[2]]) – The bottom left position of the orthotope.

  • end (list[int[2]]) – The top right position of the orthotope.

count(self: libSps.DiskSimpleVector_2D, arg0: List[int[2]], arg1: List[int[2]]) → int

Count the number of orthotopes in the given area.

Parameters

  • start (list[int[2]]) – The bottom left position of the query area.

  • end (list[int[2]]) – The top right position of the query area.

This counts by naively iterating through all points and checking their coordinates. The main purpose of this is for runtime benchmarking.

__init__(self: libSps.DiskSimpleVector_2D, path: str = '', write_mode: bool = False) → None

Create a new vector.

Parameters

  • path (str) – Prefix path of the vector on the filesystem (a file with the ending .orthos will be created), defaults to “”.

  • write_mode (str) – Open the vector in write mode (if this is set to False no changes can be made to the vector), defaults to False.
