class libSps.RamSimpleVector_3D

Simple Vector for 3-dimensional orthotopes.

add_point(self: libSps.RamSimpleVector_3D, start: List[int[3]], end: List[int[3]]) → None

Append an orthotope to the vector

Parameters

  • start (list[int[3]]) – The bottom left position of the orthotope.

  • end (list[int[3]]) – The top right position of the orthotope.

count(self: libSps.RamSimpleVector_3D, arg0: List[int[3]], arg1: List[int[3]]) → int

Count the number of orthotopes in the given area.

Parameters

  • start (list[int[3]]) – The bottom left position of the query area.

  • end (list[int[3]]) – The top right position of the query area.

This counts by naively iterating through all points and checking their coordinates. The main purpose of this is for runtime benchmarking.

__init__(self: libSps.RamSimpleVector_3D, path: str = '', write_mode: bool = False) → None

Create a new vector.

Parameters

  • path (str) – Prefix path of the vector on the filesystem (a files with the ending .orthos will be created), defaults to “”.

  • write_mode (str) – Open the vector in write mode (if this is set to False no changes can be made to the vector), defaults to False.
